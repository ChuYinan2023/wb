# BookmarkSync Chrome 扩展

## 功能特性
- 通过插件图标快速添加当前页面书签
- 右键菜单添加链接或页面
- 支持添加标签
- 自动获取页面标题和描述

## 安装步骤
1. 打开 Chrome 浏览器的扩展管理器
2. 启用"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择本目录

## 开发说明
- `manifest.json`：插件配置文件
- `background.js`：后台服务脚本
- `popup.html/js`：插件弹窗界面
- `content.js`：网页内容脚本
